# Getting Started with Create React App

## react start

npx create-react-app ...(파일이름)

## axios 설치

npm i axios

## router 설치

npm i react-router-dom

## react-player 설치

npm i react-player

## scss 설치

npm i node-sass (설치 안될 시) --> npm i sass

## react icon 설치

npm install react-icons --save
https://react-icons.github.io/react-icons
